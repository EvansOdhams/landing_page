My Task Manager

Project Description

I was inspired to create this task manager after struggling to stay organized while working on various projects at Holberton School. As a full-time student juggling multiple assignments, I found I needed a better system to track my tasks and deadlines.

This project is a portfolio assignment for Holberton School's Full Stack Software Engineering program in November 2023. I built this task manager using HTML, CSS, and JavaScript.
